# ✈️ 旅遊記帳系統 — 啟動說明

## 需要準備的 API 金鑰

### 1. Anthropic API Key（Claude AI 辨識收據用）
- 前往：https://console.anthropic.com/
- 點 API Keys → Create Key
- 複製 `sk-ant-...` 開頭的金鑰

### 2. Notion Integration Token（存資料用）
- 前往：https://www.notion.so/profile/integrations
- 點「+ New integration」
- 填名稱（如「旅遊記帳」）→ Submit
- 複製 `secret_...` 開頭的 token

### 3. Notion Database ID
執行 `python setup_notion.py` 後會自動產生並寫入。

---

## 安裝步驟

```bash
# 進入專案資料夾
cd 旅遊記帳系統

# 複製環境變數範本
cp .env.example .env

# 用文字編輯器打開 .env 填入金鑰
open .env

# 安裝 Python 套件
pip install -r requirements.txt

# 初始化 Notion 資料庫（只需執行一次）
python setup_notion.py

# 啟動系統
python app.py
```

打開瀏覽器前往：http://localhost:5001

---

## 使用方式

1. 填入旅程名稱（如「日本大阪 2026」）
2. 點擊上傳區域或拖曳收據圖片（手機可直接拍照）
3. AI 自動辨識金額、店名、類別
4. 數據即時同步至 Notion
5. Dashboard 自動更新圖表統計
